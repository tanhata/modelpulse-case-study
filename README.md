# ModelPulse Case Study

A UX case study documenting the design of ModelPulse, an ML-native monitoring platform that reduced incident detection time from 5.2 days to 30 minutes.

## Live Demo
[View Case Study](https://tanhata.github.io/modelpulse-case-study)

## About This Project

ModelPulse is a redesign of ML observability tooling for machine learning engineers. This case study documents 8 months of research, design iteration, and outcomes.

### Key Results
- **97% reduction** in time to detection (5.2 days → 30 min)
- **77% reduction** in tools per investigation (8.3 → 1.9)
- **84% reduction** in false positive rate (73% → 12%)
- **SUS Score** improved from 34 to 89
